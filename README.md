# RxCoreData

## Requirements

* Xcode 9.0
* Swift 4.0

## Installation

RxCoreData is available via [SPM](https://github.com/TheArchitect123/RxCoreData.git).

### SPM

To install RxCoreData via CocoaPods, simply add the following line to your Podfile:

```ruby
pod "RxCoreData", "~> 1.0.0"
```
